# Microservice Starter

A production-grade starter for a single microservice with selectable presets and capsules. Aimed at fast delivery with strong guardrails.

## Usage (phase one)

- Pick a preset under `presets/`
- Select capsules under `capsules/`
- Use `cookiecutter/` to generate a new service repo

## Presets

- rest: Spring MVC based service
- event: event-driven service with messaging
- reactive: Spring WebFlux based service

## Capsules

- security
- observability
- db
- messaging
- discovery
- gateway
