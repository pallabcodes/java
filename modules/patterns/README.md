# OOP Design Patterns

This module is dedicated to practicing and reviewing Object-Oriented Programming (OOP) Design Patterns.

## Directory Structure

The patterns are implemented in `src/main/java/com/backend/designpatterns`.

### Categories

1.  **Creational Patterns**
    *   Singleton
    *   Factory Method
    *   Abstract Factory
    *   Builder
    *   Prototype

2.  **Structural Patterns**
    *   Adapter
    *   Bridge
    *   Composite
    *   Decorator
    *   Facade
    *   Flyweight
    *   Proxy

3.  **Behavioral Patterns**
    *   Chain of Responsibility
    *   Command
    *   Interpreter
    *   Iterator
    *   Mediator
    *   Memento
    *   Observer
    *   State
    *   Strategy
    *   Template Method
    *   Visitor

## How to use

*   Navigate to the specific package for the pattern you are studying.
*   Review the implementation code.
*   Check the corresponding tests in `src/test/java/com/backend/designpatterns` to see usage examples.
