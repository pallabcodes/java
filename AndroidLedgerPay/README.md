# AndroidLedgerPay

Kotlin Android app scaffold with Jetpack Compose and Hilt.

## Requirements
- Android Studio Ladybug or newer
- JDK 17

## Run
1. File → Open → select `AndroidLedgerPay/`
2. Sync Gradle
3. Run `app`
