Attachments storage

Providers
- local: stores files under attachments.storage.baseDir
- s3: S3-compatible via MinIO client

Config (application.properties or env)
attachments.storage.provider=s3
attachments.storage.s3.endpoint=http://localhost:9000
attachments.storage.s3.accessKey=minioadmin
attachments.storage.s3.secretKey=minioadmin
attachments.storage.s3.bucket=attachments

Switch back to local
attachments.storage.provider=local
attachments.storage.baseDir=/tmp/attachments

Notes
- Signed URLs are generated by server; for s3 provider we still serve via download endpoint to enforce tenancy and token validation.
