# Monolithic Starter

A classic layered monolith starter with optional capsules. Use this when a single deploy without internal module boundaries is sufficient and speed is paramount.

## Usage (phase one)

- Pick layered preset under `presets/`
- Select capsules under `capsules/`
- Use `cookiecutter/` to generate a new project

## Preset

- layered: api to service to repository

## Capsules

- security
- multitenancy
- observability
- db
- messaging
