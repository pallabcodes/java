Purpose

This directory hosts external reference materials only. These are not part of the runtime build. Use them to cross-validate patterns, code structure, and operational practices.

Layout

- netflix/: Shallow clones of select Netflix OSS repositories to study production-grade practices.
- sql/: Optional reference-only DDL or queries not used by migrations.
- frontend/: Any reference UI repos or prototypes.

Notes

- Do not import or depend on these repos from application code.
- Use them to inform decisions; do not copy blindly.
